# Epage Cli

[epage](https://github.com/didi/epage) cli tool

## Usage

```
npm i epage-cli -g

epage init myEpage

# or select template
epage init myEpage --tpl vant
```

**注意**

`epage init <projectName>` 后面可跟参数

- `--tpl`：选择初始化模板，不带此参数默认`iview`模板(PC)，移动端渲染组件可附带参数`vant`
